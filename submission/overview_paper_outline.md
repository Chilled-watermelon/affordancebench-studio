# Overview Paper Outline

## Working Title

AffordanceBench Studio: An Open Toolkit for Query Processing, Evaluation, and Visualization in 3D Multimedia Affordance Research

## 1. Introduction

- software gap in 3D multimedia affordance research
- fragmented scripts and poor buildability
- toolkit contribution instead of a new method contribution

## 2. Software Overview

- dataset adapters
- query utilities
- evaluation helpers
- visualization helpers
- env-check / reproducibility support

## 3. Buildability and Usage

- CLI
- environment variables
- smoke path
- documentation

## 4. Impact and Limitations

- improves usability and reproducibility
- does not bundle large datasets by default
- does not require releasing under-review main-paper weights
