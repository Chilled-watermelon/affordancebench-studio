## Reviewer Goal

Reviewers should be able to confirm, quickly and without private context:

- what the package is
- what is intentionally included and excluded
- how to install and inspect it
- what minimal commands and outputs are available
- what independent evidence shows it builds and runs outside the authors' workstation
